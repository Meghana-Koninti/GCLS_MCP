# Google Classroom MCP Server

A production-ready **Model Context Protocol (MCP)** server that exposes
Google Classroom operations as AI-consumable tools using OAuth 2.0.

## Features
- Stateless OAuth 2.0 authentication
- Google Classroom API integration
- Role-based access enforced by Google Classroom
- MCP-compatible tool execution endpoint
- Secure environment-based configuration

## Tech Stack
- Node.js
- Express.js
- Google APIs (Classroom)
- OAuth 2.0
- MCP-style tool interface

## Setup

```bash
npm install
node index.js
